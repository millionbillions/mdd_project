# Vietnamese Mispronunciation Detection and Diagnosis

Research code for a Prompt-FiLM CTC system for Vietnamese
mispronunciation detection and diagnosis.

## Repository policy

Datasets, audio, private/public evaluation metadata, model checkpoints,
predictions, submissions, documents, slides, and credentials are intentionally
excluded from Git.

The training notebook expects datasets to be attached separately through the
runtime environment. Do not commit labeled evaluation files or generated
competition submissions.
